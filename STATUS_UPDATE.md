# 维修管理系统 - 状态更新说明

## 状态系统

### 维修状态
- **待维修**：设备已接收，等待开始维修
- **未维修**：设备无法维修或暂不维修
- **检修中**：设备正在维修过程中
- **已维修**：设备维修完成
- **报废**：设备无法修复，需要报废处理

### 状态标签样式
系统根据状态名称自动应用渐变色标签，带状态指示点：

| 状态 | 样式 | 说明 |
|------|------|------|
| 待维修 | 黄色渐变，深色文字 | 等待处理的设备 |
| 未维修 | 灰色渐变，深色文字 | 暂不维修的设备 |
| 检修中 | 青色渐变，深色文字 | 正在维修的设备 |
| 已维修 | 绿色渐变，深色文字 | 维修完成的设备 |
| 报废 | 红色渐变，深色文字 | 无法修复的设备 |

## 完成时间管理

### 手动设置完成时间
- **独立设置**：完成时间与状态更新完全独立
- **灵活控制**：可以随时设置、修改或清除完成时间
- **时间格式**：支持日期和时间选择器
- **清除选项**：可以清除已设置的完成时间

### 完成时间显示规则
- **有完成时间**：绿色显示，格式为"日期 时间"
- **无完成时间**：黄色斜体显示"未记录"
- **状态无关**：完成时间不再自动关联状态变化

## 操作功能

### 状态管理
- **状态更改**：点击铅笔图标打开状态选择弹窗
- **状态选择**：下拉菜单选择新的维修状态
- **实时更新**：状态更改后立即刷新显示

### 完成时间管理
- **时间设置**：点击时钟图标打开时间设置弹窗
- **时间选择**：日期时间选择器，默认当前时间
- **清除时间**：复选框选项清除完成时间
- **独立更新**：不影响维修状态

### 操作按钮
每条记录包含三个操作按钮：
1. **查看详情**：查看维修记录详细信息
2. **更改状态**：修改维修状态
3. **设置完成时间**：设置或清除完成时间

## 主题系统

### 5种内置主题
右下角调色板按钮一键切换，选择自动保存到浏览器：
- **科技蓝**（默认）：靛蓝紫渐变
- **温馨橙**：暖橙渐变
- **阳光金**：金黄渐变
- **少女粉**：粉色渐变
- **护眼绿**：绿色渐变

### 主题实现
- 使用 CSS 自定义属性（CSS Variables）
- 所有颜色、阴影、渐变均通过变量控制
- 图表颜色随主题变化
- 切换即时生效，无需刷新

## 弹窗系统

### 统一风格
所有弹窗采用统一设计：
- 圆角16px + 渐变头部
- 紧凑内边距，不占过多屏幕空间
- 状态弹窗和完成时间弹窗使用 `modal-sm` 尺寸
- 自定义 Alert/Confirm 弹窗同样风格

## 数据排序

- 维修记录按维修编号数字部分倒序排列
- 例如 WX-001, WX-002, ..., WX-100 按 100, 99, ..., 2, 1 排列
- 兼容 MySQL 和 SQLite 两种数据库
