# 维修管理系统 - 响应式设计说明

## 响应式设计特性

### 移动端优化
- **自适应布局**：支持手机、平板、电脑各种屏幕尺寸
- **触摸友好**：按钮大小适合手指操作
- **移动导航**：折叠式导航菜单
- **字体优化**：移动端字体大小适中

### 响应式断点
- **大屏幕**：≥1200px (lg) - 完整布局
- **中等屏幕**：≥768px (md) - 标准布局
- **小屏幕**：≥576px (sm) - 紧凑布局
- **超小屏幕**：<576px (xs) - 移动布局

## 界面适配

### 导航栏
- **桌面端**：水平排列的导航链接，渐变背景
- **移动端**：汉堡菜单 + 折叠式导航
- **品牌标识**：响应式字体大小

### 统计卡片
- **桌面端**：4列并排显示，渐变背景 + 大图标
- **平板端**：2列显示
- **手机端**：单列堆叠显示

### 主题切换器
- **位置**：右下角固定浮动按钮
- **桌面端**：48px圆形按钮
- **移动端**：40px圆形按钮
- **交互**：点击展开下拉菜单选择主题

### 数据表格
- **水平滚动**：小屏幕下支持左右滚动
- **字体调整**：移动端字体稍小
- **按钮优化**：移动端按钮紧凑排列

## UI设计规范

### 颜色系统
所有颜色通过 CSS 自定义属性控制，5种主题各有独立配色：
- `--primary` / `--primary-light` / `--primary-dark`：主色调
- `--bg` / `--bg-card`：背景色
- `--text` / `--text-muted`：文字色
- `--border`：边框色
- `--gradient`：渐变色

### 圆角规范
- **大圆角**：16px（模态框、大卡片）
- **标准圆角**：12px（卡片、表格容器）
- **小圆角**：8px（按钮、输入框、小卡片）
- **药丸形**：20px（状态标签、筛选按钮）

### 阴影规范
- **卡片阴影**：`0 4px 24px rgba(0,0,0,0.08)`
- **悬浮阴影**：`0 8px 32px rgba(primary,0.15)`
- **弹窗阴影**：`0 16px 48px rgba(0,0,0,0.2)`

### 动画规范
- **过渡时间**：0.3s cubic-bezier(0.4, 0, 0.2, 1)
- **入场动画**：fadeInUp 0.4s ease-out
- **悬浮效果**：translateY(-1px) + 阴影增强

## 组件样式

### 状态标签
- 药丸形圆角 + 渐变背景
- 左侧6px圆形指示点
- 字重600，紧凑内边距

### 按钮
- 渐变背景 + 悬浮阴影
- 悬浮时上移1px + 阴影增强
- 圆角8px，字重600

### 弹窗
- 圆角16px + 渐变头部
- 状态/时间弹窗使用 modal-sm
- Alert弹窗紫色头部，Confirm弹窗金色头部

### 图表
- 饼图改为甜甜圈图（cutout 55%）
- 圆角柱状图（borderRadius 6px）
- 统一配色方案，白色边框分隔
